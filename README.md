# C# Collections Practice

Учебный репозиторий с реализацией алгоритмов и структур данных на C# (.NET 8).
В проекте реализована работа с динамическими массивами (`List`), хеш-таблицами (`Dictionary`) и датами.

## Реализованный функционал

### 1. To-Do List Manager
Консольный менеджер задач с полным CRUD (Create, Read, Update, Delete).
- **Tech:** `List<string>`, `TryParse`, Input Validation.
- **Features:**
  - Добавление и просмотр задач.
  - "Умное удаление": можно удалить задачу по её номеру (ID) или по тексту.
  - Защита от некорректного ввода.

### 2. PhoneBook (Телефонная книга)
Система хранения контактов с мгновенным поиском.
- **Tech:** `Dictionary<string, string>`, `in/out modifiers`, Method Decomposition.
- **Features:**
  - Поиск номера по Имени (O(1)).
  - Валидация номера телефона (только цифры, длина 11 символов).
  - Защита от дубликатов имен.

### 3. Frequency Analysis (Частотный анализ текста)
Алгоритм подсчета повторяющихся слов в предложении.
- **Tech:** `Dictionary<string, int>`, `Split`, `ToLower`.
- **Logic:** Разбивает текст на слова, игнорирует регистр и считает частоту каждого слова.

### 4. LifeTime Calculator ⏳
Калькулятор времени жизни и дат.
- **Tech:** `DateTime`, `TimeSpan`.
- **Features:**
  - Расчет прожитых дней.
  - Определение дня недели рождения.
  - Алгоритм расчета дней до следующего дня рождения (учитывает, прошел ли праздник в этом году).
